# 📖 功能详情

## 🔍 多站点聚合搜索

### 搜索机制
系统采用**四轨并行**搜索策略：

```
① 本地模型库   → 已上传的个人 STL 文件（最高优先级）
② 精选模型库   → 28 条人工整理的真实可下载链接
③ 实时爬取     → Yeggi / Thingiverse / 3D溜溜网 实时抓取
④ 站点跳转     → 4 大平台搜索页直链
```

### 支持的古建筑类型
> 亭子 · 塔 · 桥 · 牌坊 · 殿 · 庙 · 祠堂 · 四合院 · 园林 · 戏台 · 民居 · 龙 · 榫卯 · 斗拱 · 飞檐 · 徽派 · 客家 · 故宫 · 宝塔 · 廊桥 · 门楼 · 宗祠

### 语义扩展
输入"亭子"会自动扩展为"凉亭/六角亭/八角亭"等相似关键词，提升搜索召回率。

## 🧊 在线 3D 预览

### 技术实现
- **Three.js**: WebGL 3D 渲染引擎
- **STLLoader**: STL 格式加载器
- **OrbitControls**: 交互控制（旋转、缩放、平移）

### 操作方式
| 操作 | 说明 |
|------|------|
| 鼠标左键拖拽 | 旋转模型 |
| 鼠标滚轮 | 缩放 |
| 鼠标右键拖拽 | 平移 |

## 🧠 AI 智能分类

### 工作原理
```
STL 模型 → Three.js 多角度渲染 → 图像 → MobileNet 分类 → 结果
```

### 分类模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| AI 分类 | TensorFlow.js MobileNet 图像识别 | 网络畅通时 |
| 文件名分类 | 基于文件名关键词匹配 | 云端部署/AI 失败时 |

### 识别的类别
- 凉亭、塔、桥、龙、狮子
- 雕像、宫殿、庙宇、房屋
- 船、飞机、动物、玩具
- 机械、工具、器具等

### 使用方法
1. 进入「我的图书馆」
2. 点击「一键分类收纳」
3. 等待 AI 识别并重命名
4. 切换「分类视图」查看
