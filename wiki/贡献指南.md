## 💻 提交代码

### 开发环境

```bash
# 1. Fork 项目

# 2. 克隆你的 Fork
git clone https://github.com/YOUR_USERNAME/stl-crawler-viewer.git
cd stl-crawler-viewer

# 3. 创建功能分支
git checkout -b feature/your-feature-name

# 4. 安装依赖
pnpm install

# 5. 开发...
pnpm dev

# 6. 提交
git add .
git commit -m "feat: 添加新功能"

# 7. 推送到你的 Fork
git push origin feature/your-feature-name

# 8. 创建 Pull Request
```

### Pull Request 指南

#### PR 标题格式
```
feat: 新功能
fix: Bug 修复
docs: 文档更新
refactor: 代码重构
test: 测试相关
chore: 构建/工具
```

#### PR 内容模板
```markdown
## 描述
[简短描述这个 PR 做什么]

## 变更类型
- [ ] 新功能 (feat)
- [ ] Bug 修复 (fix)
- [ ] 文档更新 (docs)
- [ ] 代码重构 (refactor)
- [ ] 测试 (test)
- [ ] 其他 (chore)

## 测试
- [ ] 本地测试通过
- [ ] 已添加/更新测试用例

## 截图（可选）
[如果有 UI 变更，提供截图]

## 检查清单
- [ ] 代码符合项目规范
- [ ] 已添加必要的注释
- [ ] 已更新相关文档
```
