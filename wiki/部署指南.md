# 🚀 部署指南

## 环境要求

| 要求 | 最低版本 |
|------|----------|
| Node.js | 18+ |
| pnpm | 7+ |
| Git | 2.0+ |

---

## 🖥️ 本地开发

### 1. 克隆项目

```bash
git clone https://github.com/KeyandBoy/stl-crawler-viewer.git
cd stl-crawler-viewer
```

### 2. 安装依赖

```bash
pnpm install
```

### 3. 配置环境变量（可选）

```bash
# 复制示例配置
cp .env.example .env.local

# 编辑配置文件，填入你的 Vercel Blob Token
# BLOB_READ_WRITE_TOKEN=your_token_here
```

### 4. 启动开发服务器

```bash
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000)

---

## ☁️ 云端部署

### 方式一：Vercel 部署（推荐）

#### 步骤 1: Fork 项目
点击 [Fork](https://github.com/KeyandBoy/stl-crawler-viewer/fork) 将项目 fork 到你的 GitHub

#### 步骤 2: 导入 Vercel
1. 打开 [vercel.com](https://vercel.com)
2. 点击 **「Import Project」**
3. 选择你的 GitHub 仓库
4. 点击 **「Deploy」**

#### 步骤 3: 配置 Blob 存储（可选）
> 不配置也可以使用搜索和预览功能，只是不能上传模型

1. 部署完成后，进入项目 Dashboard
2. 点击 **「Storage」** → **「Create」** → **「Blob」**
3. 创建 Blob 存储
4. Vercel 会自动注入 `BLOB_READ_WRITE_TOKEN` 环境变量
5. 点击 **「Deployments」** → 最新部署 → **「Redeploy」**

#### 🎉 完成！

获得 `your-project.vercel.app` 访问链接

---

### 方式二：Railway 部署

#### 步骤 1: Fork 项目
同上

#### 步骤 2: 创建 Railway 项目
1. 打开 [railway.app](https://railway.app)
2. 点击 **「New Project」**
3. 选择 **「Deploy from GitHub repo」**
4. 选择你的仓库
5. Railway 自动检测为 Next.js 项目

#### 步骤 3: 配置环境变量

在项目设置中添加：

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `BLOB_READ_WRITE_TOKEN` | 你的 Vercel Blob Token | 模型云存储 |

> 💡 没有 Vercel Blob Token？[获取指南](#获取-vercel-blob-token)

#### 步骤 4: 部署
等待构建完成，获得 `.railway.app` 访问链接

---

项目也支持部署到：

## 🔧 获取 Vercel Blob Token

### 方法 1: 通过 Vercel CLI

```bash
# 安装 Vercel CLI
pnpm add -g vercel

# 登录
vercel login

# 链接项目
vercel link

# 获取 Token
vercel token consume
```

### 方法 2: 通过 Vercel Dashboard

1. 打开 [vercel.com/dashboard/tokens](https://vercel.com/dashboard/tokens)
2. 点击 **「Create Token」**
3. 填写名称，选择过期时间
4. 点击 **「Create」**
5. **⚠️ 立即复制保存，过期后无法查看**

## 🐛 常见问题

### Q: 搜索结果为空？
A: 检查网络连接，确保可以访问 Thingiverse、Yeggi 等外部站点。

### Q: 模型上传失败？
A: 确认已配置 `BLOB_READ_WRITE_TOKEN` 环境变量。

### Q: AI 分类不工作？
A: 云端部署时会自动使用文件名分类作为后备方案，确保功能可用。

### Q: 3D 预览显示空白？
A: 检查浏览器是否支持 WebGL，尝试使用 Chrome 或 Edge。
