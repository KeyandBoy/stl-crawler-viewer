# ❓ 常见问题

## 📌 基础问题

### Q1: 这个项目是做什么的？
A: STL Model Crawler & Viewer 是一款专注于中国古代建筑 3D 模型的一站式工具，可以从多个平台搜索、下载、预览和管理 STL 模型文件。

### Q2: 项目是否免费？
A: 是的，项目采用 MIT 开源许可证，可以免费使用、学习和修改。

### Q3: 支持哪些文件格式？
A: 目前支持 `.stl` 格式的 3D 模型文件。

### Q4: 可以商用吗？
A: 可以，但请注意模型本身的版权。搜索到的模型版权归原作者所有。

## 🔧 技术问题

### Q5: 本地运行需要什么环境？
A:
- Node.js 18+
- pnpm 7+
- Git

详细教程见 [部署指南](部署指南)

### Q6: 为什么 3D 预览是黑的？
A: 请检查浏览器是否支持 WebGL。推荐使用 Chrome、Edge 或 Firefox 最新版本。

### Q7: 上传模型需要付费吗？
A: 如果使用 Vercel 免费额度，每个月有 10GB 的 Blob 存储免费额度，足够个人使用。

### Q8: AI 分类功能是什么原理？
A: 
- 首先尝试使用 TensorFlow.js MobileNet 进行图像识别
- 如果失败（网络问题或云端环境），会自动切换到基于文件名的关键词匹配
- 两种方式都能正常工作，确保云端部署时功能可用

## ☁️ 部署问题

### Q9: Vercel 部署失败怎么办？
A: 常见解决方法：
1. 检查 GitHub 仓库是否正确链接
2. 查看部署日志具体报错信息
3. 确认 `package.json` 中的依赖格式正确

### Q10: Railway 和 Vercel 有什么区别？
A:
| 特性 | Vercel | Railway |
|------|--------|---------|
| 免费额度 | 100GB 带宽/月 | $5/月 |
| Blob 存储 | 原生支持 | 需配置 |
| 中国访问 | 一般 | 较好 |
| Next.js 优化 | 最佳 | 良好 |

### Q11: 如何暂停 Railway 部署避免扣费？
A:
1. 进入 Railway Dashboard
2. 选择你的项目 → Settings
3. 找到 **Danger Zone**
4. 点击 **Suspend Service**

## 🔍 搜索问题

### Q12: 搜索结果太少？
A: 
- 尝试使用更通用的关键词（如"亭子"而非"六角亭"）
- 检查网络是否正常
- 部分平台可能需要科学上网

### Q13: 有些模型无法下载？
A: 
- 部分模型需要登录 Thingiverse 账号才能下载
- 部分为付费模型
- 系统会提示"跳转查看"让你访问原站

### Q14: 如何添加更多搜索来源？
A: 
- 编辑 `src/app/api/search-stl/route.ts`
- 添加新的爬虫逻辑
- 遵循现有的代码结构
