# 🏠 首页

**STL Model Crawler & Viewer** 是一款专注于中国古代建筑 3D 模型的一站式搜索、预览与管理平台。

## ✨ 主要功能

| 功能 | 说明 |
|------|------|
| 🔍 多站点聚合搜索 | 同时检索 Thingiverse、Printables、爱给网等多个平台 |
| ⬇️ 直接下载 STL | 一键下载真实可用的 STL 文件 |
| 🧊 在线 3D 预览 | Three.js 实时渲染，支持旋转缩放 |
| 📁 个人模型库 | 上传、存储、管理你的 STL 收藏 |
| 🧠 AI 智能分类 | 自动识别模型类型并分类整理 |

## 🚀 快速开始

### 1. 搜索模型
在搜索框输入关键词（如"六角亭"、"廊桥"），即可获取多平台搜索结果。

### 2. 下载使用
- **直接下载**：点击绿色按钮获取 STL 文件
- **跳转查看**：点击蓝色按钮访问原站页面

### 3. AI 分类
进入「我的图书馆」，点击「一键分类收纳」，AI 自动识别并整理模型。
