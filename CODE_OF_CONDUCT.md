# 📜 行为准则

## 我们的承诺

为了营造一个开放、包容、友好的社区环境，我们承诺无论年龄、身体状况、残疾、种族、性别认同、经验水平、国籍、外貌、种族、宗教或性取向如何，都不会对参与者进行骚扰。

## 我们的标准

有助于创造积极环境的行为包括：

- 🗣️ 使用友好和包容性的语言
- 🤝 尊重不同的观点和经验
- 👍 优雅地接受建设性批评
- 💬 关注对社区最有益的事情
- 🙋 对其他社区成员表现出同理心

不可接受的行为包括：

- 😠 使用性暗示的语言或图像
- 🔥 进行个人或政治攻击
- 👎 公开或私下骚扰
- 📢 未经同意发布他人的私人信息
- ❌ 其他可被合理认为是不当的行为

## 我们的责任

项目维护者有责任明确和公平地解释可接受行为的标准，并应采取适当和公平的措施应对不可接受的行为情况。

项目维护者有权和责任删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、Wiki 编辑、问题和其他贡献，或临时或永久禁止任何贡献者从事他们认为不当、威胁、冒犯或有害的其他行为。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或社区的情况。代表项目或社区的例子包括使用官方电子邮件地址、通过官方社交媒体账户发布或在线或离线活动中担任指定代表。

## 执法方式

关于滥用、骚扰或其他不可接受行为的投诉可以联系项目团队。项目团队将审查和调查所有投诉，并采取必要和适当的方式回应。项目团队有义务对事件举报人保密。

## 执行方式

如果出现明显违反本行为准则的情况，项目维护者可以自行决定是否立即采取行动，无需事先警告。

## 举报方式

如果你发现违规行为，请通过以下方式联系：

- 📮 GitHub Issues
- 📧 电子邮件

所有投诉都将得到及时和公平的审查和调查。
